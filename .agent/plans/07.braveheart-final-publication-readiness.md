- CTA and stub documentation review.

2. Run `npm run preview` and test all 10 Phase 1 routes.
3. Test Navbar and Footer links on desktop and mobile (375px).

11. Record pass/fail per route in validation report.
12. File follow-up issues for any failures — do not mark Phase 1 complete if CLAUDE.md validation criteria fail.
