# Brave Heart Brand and Style Guide

## Tone

Brave Heart should sound:

- direct;
- respectful;
- protective;
- credible;
- mission-focused;
- financially clear;
- non-gimmicky.

## Visual Direction

Use the cloned listing site as a premium shell, but shift the mood from luxury property to service, trust, protection, and transaction relief.

## Color Direction

```text
Deep navy / blue-black background
Gold accent
White / warm-off-white text
Pale slate muted text
Translucent gold/white borders
Dark glass surfaces
```

## Typography

Use the cloned site's typography unless there is a clear reason to change. Prioritize readability on mobile.

## Image Style

Images should support:

- public safety identity;
- household stability;
- real transaction savings;
- secure verification;
- professional oversight;
- HomeToHome technology support.

Avoid imagery that feels generic, gimmicky, overly militarized, or unrelated to real estate outcomes.

## Copy Rules

- Prefer concrete explanations over slogans.
- Avoid unsupported guarantee language.
- Avoid exaggerated urgency.
- Do not imply legal, lender, tax, or brokerage conclusions without approved source language.
- Use plain-language descriptions of benefits and verification.
